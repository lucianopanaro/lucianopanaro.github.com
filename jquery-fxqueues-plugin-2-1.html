jQuery FxQueues plugin 2.1

Introduction

This version of jQuery FxQueues is intended for jQuery 1.4 - Bug fixes and improvements for jQuery1.3 will be reflected in 2.0.x versions. For the plugin's documentation see jQuery FxQueues 2.0 docs. You can check the code from github under the jQuery1.4 branch: http://github.com/lucianopanaro/jQuery-FxQueues/tree/jQuery1.4

Download

Until the new plugins site for 1.4 is release, you can grab the code from github: http://github.com/lucianopanaro/jQuery-FxQueues/blob/jQuery1.4/jquery.fxqueues-2.1.js
